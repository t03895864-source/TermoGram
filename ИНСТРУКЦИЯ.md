## 🖥️ СБОРКА .EXE ДЛЯ WINDOWS

### Вариант 1: Быстрая сборка (рекомендую) ⭐

```powershell

```

**Жди 5-10 минут** — появится файл:

```powershell
# 1. Очисти кэш

# 2. Пересобери node-gyp
npm install --build-from-source

# 3. Попробуй снова
npm run build:exe
```

```powershell

npm install electron electron-builder --save-dev

# 2. Проверь что electron-main.js существует
# (должен быть в папке проекта)

# 3. Собери
npx electron-builder --win --x64
```

## 📱 СБОРКА .APK ДЛЯ ANDROID

### 🚀 СПОСОБ 1: PWABuilder (БЫСТРО - 5 минут)

#### Шаг 1: Запусти TERMOGRAM

```powershell
cd C:\Users\Timur\.koda\tg-clone
npm start
```

#### Шаг 2: Установи ngrok

1. Скачай с https://ngrok.com/download
2. Распакуй в `C:\Users\Timur\.koda\tg-clone\ngrok.exe`
3. Зарегистрируйся на ngrok.com (бесплатно)
4. Скопируй свой токен

#### Шаг 3: Запусти ngrok

**Новое окно PowerShell:**
```powershell
cd C:\Users\Timur\.koda\tg-clone
.\ngrok.exe http 3000
```

#### Шаг 4: Скопируй ссылку

Увидишь что-то вроде:
```
Forwarding  https://a1b2-c3d4.ngrok.io -> http://localhost:3000
```

**Скопируй https://....ngrok.io**

#### Шаг 5: Иди на PWABuilder

1. Открой https://pwabuilder.com
2. Вставь ссылку ngrok в поле URL
3. Нажми **Analyze**
4. Нажми **Build for Android**
5. Скачай **APK** файл

#### Шаг 6: Установи на телефон

1. Перекинь APK на телефон
2. На телефоне: Настройки → Безопасность → Разрешить установку из неизвестных источников
3. Запусти APK → Установить

### 🛠️ СПОСОБ 2: Capacitor (для продвинутых)

#### Шаг 1: Установи Capacitor

```powershell
cd C:\Users\Timur\.koda\tg-clone
npm install @capacitor/core @capacitor/cli @capacitor/android --save-dev
```

#### Шаг 2: Инициализируй

```powershell
npx cap init
```

Введи:
- App name: `TERMOGRAM`
- App ID: `com.termogram.app`

#### Шаг 3: Настрой capacitor.config.json

```json
{
  "appId": "com.termogram.app",
  "appName": "TERMOGRAM",
  "webDir": "public",
  "server": {
    "url": "https://твоя-ссылка-ngrok.io"
  }
}
```

#### Шаг 4: Добавь Android

```powershell
npx cap add android
npx cap sync
```

#### Шаг 5: Открой в Android Studio

```powershell
npx cap open android
```

#### Шаг 6: Собери APK в Android Studio

1. Подожди пока проект загрузится (5-10 мин)
2. Menu: **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
3. Жди сборки
4. APK будет в: `android\app\build\outputs\apk\debug\app-debug.apk`

## ❌ ЧАСТЫЕ ОШИБКИ

### Ошибка: "sqlite3 не собирается"

**Решение:**
```powershell
npm cache clean --force
Remove-Item -Recurse -Force node_modules
npm install
npm run build:exe
```

### Ошибка: "ngrok не запускается"

**Решение:**
1. Зарегистрируйся на https://ngrok.com
2. Скачай токен
3. Запусти: `.\ngrok.exe config add-authtoken ТВОЙ_ТОКЕН`
4. Попробуй снова: `.\ngrok.exe http 3000`

### Ошибка: "PWABuilder не видит сайт"

**Решение:**
1. Убедись что ngrok работает
2. Проверь что ссылка https:// (не http://)
3. Обновить ссылку в PWABuilder
4. Ngrok меняет ссылку при каждом запуске!

### Ошибка: "APK не устанавливается"

**Решение на телефоне:**
1. Настройки → Безопасность
2. Включи "Неизвестные источники"
3. Попробуй снова

### Ошибка: "electron-builder не найден"

**Решение:**
```powershell
npm install electron-builder --save-dev
npm run build:exe
```
